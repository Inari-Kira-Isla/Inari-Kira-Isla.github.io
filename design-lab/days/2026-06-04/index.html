§ 04踩坑與教訓 / Hotfix Log

四個陷阱，
四次根因斷案。

HOTFIX 01

PostgREST 雙編碼

根因 requests.params 二次 encode → 404
修法 直接拼 URL 繞 params，4 腳本 audit 修 3 fn
擴散 已建議 weekly Hermes lint job 防回歸
HOTFIX 02

trust_score_audits CHECK 過嚴

根因 entity_type 缺 openclaw_article → 23514 reject
修法 Migration apply 4 / 4 PASS（含 INDUSTRY 白名單擴張）
擴散 CHECK constraint 8 → 12 entity type
HOTFIX 03

AEO 3 新 fn signature

根因 單參數呼叫 raise — 缺 default 推導
修法 加 default + 自動推導，3 fn 全綠
擴散 dispatcher 700 行 wire-up 後 50 篇 baseline 通過
HOTFIX 04

Cloudflare WAF UA

根因 api.supabase.com 第 2-3 req 起 403
修法 強制 Mozilla browser-like UA
擴散 PostgREST .supabase.co 不受影響，已記入 MEMORY.md
